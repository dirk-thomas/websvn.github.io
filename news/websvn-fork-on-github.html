WebSVN was forked on GitHub (Current)

The official (original) WebSVN project is no longer active and was last released on June 27, 2011. This fork exists as a replacement, and changes include the addition of PHP 7 support.

This website (and documentation) originated from the original project and may contain outdated information. Please file an issue on GitHub if you find a problem.
