WebSVN 2.2.1 released! (05/11/2009)

WebSVN 2.2.1 has been released to fix some issues of the previous version.
It is available from websvnphp.github.io/download/

This release fixes an issue when using the gz-extension to compress archives instead of the default php-based implementation. Additionally the output in case of an error with the svn command line client has been improved.
