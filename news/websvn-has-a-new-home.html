WebSVN has a new home (11/07/2008)

The official website of WebSVN has moved to www.websvn.info! The new website will contain more information about the project and will be used for a better dialog with the users of WebSVN.

In addition to the website, we are working on a new version of WebSVN, which will contain all patches and fix some security issue. So stay tuned for more news!
