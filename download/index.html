Access subversion repository

You can also access the subversion repository directly. The repository is located under http://websvn.tigris.org/svn/websvn/, the username is 'guest' without a password. The stable releases can be found in the folder 'tags', whereas 'branches' and 'trunk' contain unstable development versions. Uses these versions only in a trusted environment!
